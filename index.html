Fruit Collector!

Try to guess the number by collecting different fruits!

At the start of the game you will be given a random number.

There are four different fruits below, each with a specific point total.

You can win the game by matching the random number you are given to your point total. However, if you go above the random number you are given, you lose.

The value of each fruit is hidden to you until you click on it.

Each time the game starts, the point value of each fruit will change.
